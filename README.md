# npm-import

## Importing code

Clone this repo
```bash
git clone https://github.com/mozey/npm-import.git
cd npm-import
```

Install node_modules directly from [mozey/npm-package](https://github.com/mozey/npm-package)<sup>[1]</sup>
```bash
npm install
```
